安装

安装命令：
npm install -g edp

-g为全局安装参数，必选。
在Linux/Mac平台下，全局安装可能需要sudo。

包管理

导入包命令：
edp import <package>

导入包的指定版本：
edp import <package>@<version>

项目管理

项目初始化：
edp project init

将当前目录初始化为项目目录
项目目录不允许嵌套

项目构建

构建命令：
edp build

指定构建输出目录：
edp build --output <path>
